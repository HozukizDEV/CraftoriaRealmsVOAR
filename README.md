## Sobre o Plugin

Este plugin foi desenvolvido **exclusivamente** para o servidor **CraftoriaRealms**, com o objetivo de aprimorar a experiência dos jogadores. Cada detalhe foi cuidadosamente pensado para tornar a jogabilidade mais divertida, dinâmica e personalizada.  

Compatível com **Minecraft 1.8.8**, garante estabilidade e integração completa com os sistemas já existentes no servidor.

---

## Funcionalidades Principais

- Suporte a traduções e personalizações via arquivos `.yml`.  
- Comandos customizados exclusivos para o servidor.  
- Integração total com eventos do servidor, permitindo interações avançadas.  
- Compatível com futuras atualizações do servidor (com pequenos ajustes).  
- Pequenos easter eggs para jogadores curiosos descobrirem!  

---

## Requisitos

- Servidor Minecraft versão **1.8.8**.  
- Java 8 ou superior instalado no servidor.  
- Permissão para adicionar plugins na pasta `/plugins`.  

---

## Instalação

1. Faça o download do plugin (arquivo `.jar`).  
2. Coloque o arquivo dentro da pasta `/plugins` do seu servidor.  
3. Reinicie o servidor.  
4. Verifique o console para garantir que o plugin foi carregado corretamente.
